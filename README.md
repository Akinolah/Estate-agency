# Estate Agency - Firebase Studio Project

This is a Next.js real estate application starter generated by Firebase Studio.

## Features

- Property Listings Display & Filtering (Nigeria Focused)
- Interactive Maps (Requires Google Maps API Key)
- Virtual Tours Section
- Mortgage Calculator (Advanced with PITI & Amortization)
- Customer Reviews Carousel
- Contact Form & Information
- Gallery Room (Masonry Layout)
- Responsive Design
- Multi-Currency Support (USD, NGN, GBP - Placeholder rates)
- Back-to-Top Button
- Accessibility & Legal Pages (Privacy, Terms)

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone <your-repository-url>
    cd <your-project-directory>
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Set up Environment Variables:**

    -   Create a file named `.env.local` in the root of your project. **Do not** commit this file to version control.
    -   Add the following environment variables to your `.env.local` file:

        ```dotenv
        # Google Maps API Key (Required for Interactive Maps)
        # Obtain a key from Google Cloud Console: https://console.cloud.google.com/google/maps-apis/credentials
        # - Ensure "Maps JavaScript API" and "Places API" are ENABLED for this key.
        # - Ensure Billing is enabled on your Google Cloud project.
        # - Restrict the key (e.g., to your domain) for security.
        # - If you encounter 'InvalidKeyMapError' or 'BillingNotEnabledMapError', double-check these steps, the key itself, and ensure billing is active for the project.
        NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=YOUR_MAPS_API_KEY_HERE

        # Optional: Google Maps Map ID (For custom map styling)
        # Create a Map ID in Google Cloud Console: https://console.cloud.google.com/google/maps-apis/studio/maps
        # NEXT_PUBLIC_GOOGLE_MAPS_MAP_ID=YOUR_MAP_ID_HERE
        ```

    -   **Important:** Replace `YOUR_MAPS_API_KEY_HERE` with your actual, **valid**, and correctly configured Maps API key. Ensure billing is enabled for the associated Google Cloud project.

4.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

5.  Open [http://localhost:9002](http://localhost:9002) (or the specified port) with your browser to see the result.

## Project Structure

-   `src/app/`: Contains the main application pages using Next.js App Router (e.g., Home, About, Listings, Services, Contact, FAQ, Legal pages).
-   `src/components/`: Reusable UI components.
    -   `src/components/ui/`: Components from shadcn/ui.
    -   `src/components/layout/`: Header and Footer components.
    -   `src/components/providers/`: Context providers (Maps, QueryClient, Currency).
    -   `src/components/services-page-components/`: Components specific to the Services page.
    -   Other functional components (e.g., PropertyCard, HeroSlider, MortgageCalculatorSection).
-   `src/lib/`: Utility functions (e.g., `cn` for class names).
-   `src/hooks/`: Custom React hooks (e.g., `useToast`, `use-mobile`, `useCurrency`).
-   `src/services/`: Business logic services (e.g., `mortgage-calculator.ts`).
-   `src/types/`: TypeScript type definitions (e.g., `property.ts`).
-   `public/`: Static assets.
-   `src/app/globals.css`: Global styles and Tailwind CSS theme configuration.

## Customization

-   **Styling:** Modify `src/app/globals.css` to adjust the Tailwind CSS theme and shadcn/ui variables (colors, radius, etc.).
-   **Content:** Replace placeholder data in page components (e.g., `src/app/page.jsx`, `src/app/listings/page.jsx`, sample addresses, phone numbers, currency rates in `currency-provider.jsx`) with your actual data source (API, database).
-   **API Keys:** Ensure you have a valid and correctly configured API key set up in your `.env.local` file for Google Maps, and that billing is enabled for the associated Google Cloud project. See **Getting Started** for details.
-   **Currency:** Update exchange rates in `src/components/providers/currency-provider.jsx` with real-time data or more accurate static rates. The base currency is currently USD.
-   **Contact Info:** Update addresses, phone numbers, social links, and email addresses in `src/app/contact/page.jsx` and `src/components/layout/footer.jsx`.
-   **Property Data:** Update the `sampleProperties` array in relevant page files (`page.jsx`, `listings/page.jsx`, `services/page.jsx`) or replace it with data fetching logic. Ensure property data matches the structure used in components like `PropertyCard`.
